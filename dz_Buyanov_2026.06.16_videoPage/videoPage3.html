🎬 Поиск фильмов, сериалов и эпизодов

📌 Как это работает Режим Что делает 🎬 Фильмы Ищет фильмы по названию, показывает рейтинг, жанры, год 📺 Сериалы Ищет сериалы, показывает количество сезонов + кнопка "Показать эпизоды" 📺 Эпизоды Ищет сериалы, затем ищет внутри эпизодов по названию и показывает совпадения 🎯 Особенности Фильмы — показываются с рейтингом и жанрами Сериалы — показываются с количеством сезонов, можно посмотреть эпизоды Эпизоды — ищет по названию внутри всех серий найденных сериалов Теперь у тебя полноценный поиск с тремя режимами! 🚀
